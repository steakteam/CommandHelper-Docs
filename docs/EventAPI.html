이벤트 API 문서

이벤트는 스크립트를 명령어를 치지 않아도 실행할 수 있게 해줍니다. 예를 들어 플레이어 로그인, 블록 부수기 등이 있습니다. 초보자 가이드의 이벤트 항목을 통해 더 많은 정보를 얻으세요.
